# Skills

Standalone Agent Skills maintained by CodeCrafters Automation.

Each skill lives in its own subdirectory here (e.g. `skills/<skill-name>/SKILL.md`). Use this directory for skills that are **not** tied to a specific plugin — reusable building blocks you might copy into a project's `.claude/skills/` or fold into a plugin later.

Skills that ship as part of a plugin belong inside that plugin instead, under [`../plugins/<plugin>/skills/`](../plugins/).
