# Plugins

Claude Code plugins maintained by CodeCrafters Automation. Each plugin lives in its own subdirectory here and is cataloged in [`../.claude-plugin/marketplace.json`](../.claude-plugin/marketplace.json), where its `source` points to `./plugins/<folder>`.

| Plugin | Folder | Install |
|--------|--------|---------|
| Change Claude Profile (`ccp`) | [`ccp/`](ccp/) | `/plugin install ccp@codecraftersautomation` |

Each plugin's own README (e.g. [`ccp/README.md`](ccp/README.md)) documents its purpose and how to use it.
